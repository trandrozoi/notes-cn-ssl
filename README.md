# notes-cn-ssl

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面，内容主要围绕 SSL/TLS 相关的学习笔记、参考文档和实用工具页面。这些页面不针对任何特定域名或具体网站，而是作为通用参考资料存放于此。

## 目录说明

```
notes-cn-ssl/
├── docs/               # 存放所有 HTML 页面文件
│   ├── index.html      # 首页入口
│   └── ...             # 其他独立页面
├── assets/             # 静态资源（CSS、JS、图片等）
└── README.md           # 本文件
```

- **docs/**：所有可发布的 HTML 页面均放置于此目录，可通过 GitHub Pages 或其他静态托管服务直接访问。
- **assets/**：存放页面所需的样式表、脚本、字体、图标等静态文件，按需组织子目录。

## 页面归档说明

- 每个 HTML 文件均为独立页面，彼此之间无强依赖关系。
- 页面内容侧重于 SSL/TLS 概念解析、配置示例、常见问题梳理等通用技术主题。
- 不包含任何指向具体商业网站、服务或产品的链接或引用。
- 页面以中文撰写，部分术语保留英文原文以便查阅。

- 欢迎提交 Issue 报告错误或提出改进建议。
- Pull Request 请确保新增或修改的页面内容保持独立、清晰，并遵循已有的文件结构。
